# 💼 WebStudio Landing Page

A modern responsive landing page for a web studio, built with semantic HTML and CSS.  
The project showcases a company’s services, team, and contact form.

---

## 🔗 Live Demo

👉 https://katarginakate.github.io/goit-markup-hw-06/

---

## 📖 Project Overview

This project represents a **responsive website for a digital agency (WebStudio)**.

It includes:

- Hero section with call-to-action
- Features / advantages section
- Services overview
- Team section
- Clients section
- Subscription & contact form

The layout is fully responsive and adapted for different screen sizes.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox)
- Responsive Design
- JavaScript (ES6+)
- Git & GitHub

Or use Live Server in VS Code.
